# Nest-Toolbox

---

### Help

```
nest --help
```

### Install Nest CLI

- generate files
- run app
- compile app
- bundle app

```
npm i -g @nestjs/cli
```

### Create new project

```
nest new
```

### Run server

```
nest start --watch
```

### Create new controller

```
nest g co

```

or with no test file

```
nest g co --no-spec

```
